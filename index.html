1. INFORMATION COLLECTED

When a user elects to create an account, the application may collect and process the following categories of information:

  • Email address (used for authentication and account access)
  • Authentication credentials (securely managed via Firebase Authentication)
  • Unique user identifier assigned by the authentication system
  • Gameplay data, including scores and progression metrics
  • Virtual in-app currency balances (“satoshi points”)
  • Withdrawal request submissions initiated by the user

In addition, the application provides a “Play for Fun” mode, which allows users to access core gameplay functionality without account creation and without the collection of personal data.

The application does not knowingly collect real-world identity data such as legal names, physical addresses, or geographic location data.

All authentication and related services are facilitated through third-party infrastructure provided by Firebase.

2. PURPOSE OF DATA PROCESSING

  • User account creation and authentication
  • Maintenance of gameplay functionality and progression tracking
  • Management of virtual in-app currency balances
  • Processing and logging of withdrawal requests
  • Detection and prevention of fraudulent activity, abuse, or manipulation
  • General improvement and maintenance of application performance

The application includes a virtual scoring and reward system utilizing units referred to as “satoshi points.”

For the avoidance of doubt:

  • Satoshi points are strictly virtual in-app units
  • Satoshi points do not constitute currency, legal tender, or cryptocurrency
  • Satoshi points carry no intrinsic or guaranteed real-world value
  • Satoshi points are provided solely for entertainment and gameplay progression

Nothing within the application shall be interpreted as establishing an obligation to exchange, redeem, or convert virtual points into real-world value.

4. WITHDRAWAL REQUESTS AND REDEMPTION PROCESS

The application includes a withdrawal request function enabling users to submit requests relating to accumulated virtual points.

Such requests are transmitted via backend systems operated through Firebase and are reviewed externally by the developer.

For clarity and legal effect:

  • Submission of a withdrawal request constitutes a non-binding notification only
  • No automatic redemption, cash-out, or in-application payout mechanism exists
  • Submission of a request does not create any entitlement, expectation, or guarantee of reward
  • All determinations regarding approval, denial, timing, and method of any discretionary payout are made solely at the developer’s discretion

The application does not provide financial services, investment products, or cryptocurrency exchange services.

5. PLAY FOR FUN MODE

The application provides a “Play for Fun” mode intended for unrestricted gameplay.
