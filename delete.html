Account & Data Deletion Request

This application does not require user accounts and does not provide a login system. Therefore, no account deletion is required.

Data Collection

This application does not directly collect or store any personal user data on external servers.

The application uses Google AdMob, which may process limited device-level information for advertising purposes, such as advertising identifiers and general device information.

Local Data

This application stores gameplay data locally on the user’s device only. This includes:

  • Game progress
  • Scores
  • In-game points (stored locally only)

This data is not transmitted to or stored on external servers.

Data Deletion Requests

Because no accounts or server-side user data are collected or stored, there is no centralized data available for deletion by the developer.

Users may delete all locally stored data at any time by clearing the app data or uninstalling the application from their device.
