Account & Data Deletion Request

This page explains how users may request deletion of their account and/or associated data.

How to Submit a Request

Users may request deletion of their account, personal data, or both by contacting:

Email: youractualemail@gmail.com

Please include the email address used in the application so we can identify the correct account.

Types of Deletion Requests

  • Account Deletion: Permanently removes the user account and login credentials.
  • Data Deletion: Removes user-associated data including gameplay progress, virtual currency (“satoshi points”), and stored records.
  • Full Deletion: Removes both account and all associated data.

Processing Time & Retention Policy

Deletion requests are processed manually through backend systems operated via Firebase services.

Once a deletion request is submitted, it will be scheduled for processing. For safety and fraud prevention purposes, a 3-day retention period is applied before permanent deletion.

During this 3-day period, users may contact us to cancel or reverse a deletion request if it was made in error.

After the retention period has passed, the data will be permanently deleted and cannot be recovered.

Data Stored

  • Email and authentication data
  • Game progress and scores
  • Virtual in-app currency (“satoshi points”)
  • Withdrawal request history
  • Any associated records stored in Firebase databases

Third-Party Services

User data is stored and managed using Firebase backend services.

All deletion operations include removal from connected Firebase Authentication and database systems.
