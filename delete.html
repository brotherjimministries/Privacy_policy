Account & Data Deletion Request

This page explains how users may request deletion of their account and associated data.

How to Request Deletion

Users may request deletion of their account and all associated data by contacting:

Email: brotherjimministries@gmail.com

Please include the email address used to register the account in your request so we can locate the correct data.

Data That Will Be Deleted

  • Account login information
  • Gameplay progress and scores
  • Virtual in-app currency data (“satoshi points”)
  • Withdrawal request history
  • Any associated records stored in backend systems

Processing Time

Deletion requests are processed manually and will be completed within a reasonable timeframe.

Backend Services

User data is stored using Firebase services and will be removed from all associated databases upon request.
